So, for the last two days, I basically worked on how I could use the ideas in LeJEPA or LeWMJEPA to avoid using a student-teacher EMA setup.
The idea basically now is: One encoder, two losses(pred + SIGREG). I understood thoroghly how SIGREG works and how to implement it.
The idea is that you project multi dim latents(sampled from encoder embedding space), onto many unit vectors of the same dimensionality. 
// This uses something called the Cramer Wold principle which tells us that a higher dimensional multivariate distribution can be adequately "represented"(for testing purposes), by many 1-D unit dim projections i.e. we can test it's 1-D projections for normality and use that to know how close to a Gaussian the original encoder distribution is.
Then, the projections of these latents are tested for normality using the Epps-Pulley statistic, it uses the empirical characteristic(1/N * sum_from_1_to_N(exp(i*u^T*h), where h is the projected latent)). and N is number of latents
The diff is then squared and integrated across a shit ton of values: Int((empirical - normal Gaussian characteristic fn)^2 dt)
This is then averaged across the number of unit dim vectors, and using this we get our SIGREG loss.
